# A.L.E.C - Adaptive Learning Executive Coordinator

**Your Personal AI Companion - Trained on Your Data, Language, and Thought Processes**

*Like JARVIS from Iron Man - but smarter, more personal, and constantly learning*

## Features
- 🧠 **35B Parameter LLM Base** (Llama 3.1 or Mistral Large)
- 💬 **Natural Voice Interface** with real-time speech-to-text
- 📚 **Personal Data Training** on your emails, texts, documents
- 🔒 **Secure Token System** - Separate tokens for STOA access and full capabilities
- 🤖 **Adaptive Learning** - Learns from interactions, improves over time
- 🏠 **Smart Home Integration** - Connects to Home Assistant, IoT devices
- 💻 **Local First** - Runs on your Mac, no cloud dependencies required
- 🚀 **Initiative & Sass** - Proactive suggestions, witty personality
